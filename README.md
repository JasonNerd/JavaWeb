## 文件夹说明
|名称|说明|
|:-|:-|
course|简要的课程笔记，包含着关键知识点|
project|以关键知识点作为核心的可运行项目，每一个项目都应说明运行方法
reports|报告或者日志，记录项目中的问题和解决方案
data|数据，可能的数据文件，例如作为输入、中间缓存或结果输出的文件
reference|任何参考的代码或者文档
